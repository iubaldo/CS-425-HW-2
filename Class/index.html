This program creates a "merry go round" made from a number of truncated cones.
Each cone is generated by calculating vertices using trigonometric functions,
which are then used to form the triangles that make up the cone. Each cone then
has a transformation applied to it that moves and rotates it to its proper spot,
along with allowing for animation.
